## 🧠 Sobre mí

Desarrollador argentino de sistemas interactivos, enfocado en producto y experiencia.

Técnico y Desarrollador Jr en Ultra Interactive, creando software y experiencias para eventos.  
Inversionista y Desarrollador en Bonemelt Studios, trabajando en proyectos como TailTales.

Stack: Java (Spigot/Paper), TypeScript, Next.js, Supabase y SQLite.

Desde los 13 años construyo servidores de videojuegos: GTA V Roleplay y múltiples servidores de Minecraft con sistemas propios.

Hoy desarrollo desde game systems hasta plataformas y software a medida, combinando arquitectura, lógica y experiencia de usuario.
